Veterinary Clinic web-application created as a frontend project with React and Bootstrap CSS

run frontend
### `npm install`
### `npm start`

run backend
### `cd pet-backend`
### `npm install`
### `npm start`

Open [http://localhost:3000] to view in browser. Back-end runs on [http://localhost:4000]
